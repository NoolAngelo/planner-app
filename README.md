## Tech Stack

- **Frontend:** React (TypeScript) + Vite
- **Styling:** Tailwind CSS
- **Data:** Browser `localStorage`

## How to Run Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open `http://localhost:5173` in your browser.
